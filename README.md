# 🎨 Color Harmony

图片配色提取工具 — 上传一张图，自动摘出它的温柔配色。

## ✨ 功能

- **图片取色** — 拖拽或上传 JPG / PNG / WEBP，自动提取 4–6 个主色调
- **收藏色号** — 喜欢的配色一键保存，支持搜索、重命名
- **云端同步** — 邮箱注册登录，配色数据跨设备自动同步
- **一键复制** — 点击色号即复制，整组配色一键带走

## 🚀 体验

👉 [wenkanghwdd.github.io/color-harmony](https://wenkanghwdd.github.io/color-harmony/)

## 🛠 技术

纯前端单文件，基于 Color Thief 取色算法，Supabase 提供后端认证与数据同步。
